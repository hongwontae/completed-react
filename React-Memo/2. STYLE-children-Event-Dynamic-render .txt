1. Style
1-1. 리액트 컴포넌트에 Style를 적용하기 위해 CSS 파일을 생성했다면
 컴포넌트 파일 안에서 CSS를 import '../kkk.css' 처럼 가져와야 적용됩니다.
1-2. 1-1 방법을 선택하면 모든 SCOPE 되지 않고 모든 컴포넌트에서 적용되기에 
 효율적인 방식은 아닙니다.


2. children props
2-1. 커스텀 컴포넌트의 열린 태그와 닫힌 태그 사이의 콘텐츠를 의미합니다.
2-2. 자식 컴포넌트가 props.children으로 받아서 사용할 수 있습니다.
2-3. children은 배열, 객체, 태그, 텍스트, 컴포넌트.. 등 자유롭게 넣을 수 있습니다.


3. React Event
3-1. 바닐라 JS는 모든 요소에 이벤트를 등록해버립니다.
=> addEventListener('click', ()=>{console.log('event occurred')})
3-2. React는 단 하나의 리스너로 모든 클릭 이벤트를 감지합니다.
=> document.addEventListener('click',()=>{...})
=> 어떤 컴포넌트에서 클릭이 발생했는지는 React가 분석해서 해당 컴포넌트의 onClick를 호출합니다.
=> 또한 언마운트가 되면 이벤트 제거도 자동으로 처리합니다.


4. JSX {} ++
4-1. JSX 내에서 중괄호는 표현식이라고 부릅니다.
4-2. 삼항 연산자나 && 연산자를 통해 조건부 렌더링을 할 수 있습니다.
4-3. && 연산자는 두 가지 모두 TRUE여야 TRUE인 특징이 있습니다.
=> {조건 && <component/>}이라면 조건이 
case1-> true인 경우에는 component가 반환
case2-> false인 경우 아무것도 반환하지 않음
