1. styled-component 동적 CSS
1-1. 상태값을 지정하고 상태 변경함수를 등록합니다.
1-2. 상태값은 특정 행동에 따라 변경 함수를 통해 상태가 변경되고 이에 대한 파생결과를 등록합니다.
1-3. 스타일 컴포넌트에 key={1-2}를 속성으로 등록합니다.
1-4. 자체 스타일 컴포넌트에서 동적값을 통해 key에 접근할 수 있습니다.
 styled.div `
    color : ${(props)=>(props.isValid ? 'blue' : 'red')};
    backgroud : ${({isValid}) => (isValid : 'white' : 'black')}
 `
